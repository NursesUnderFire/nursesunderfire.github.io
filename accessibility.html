Accessibility Statement

We want everyone to be able to access advocacy resources, regardless of technology or ability. Our goal is to meet or exceed WCAG 2.1 AA guidelines.

What We Have Implemented

  • Logical heading order, descriptive link text, and meaningful alt attributes.
  • Skip links, keyboard-focus styles, and reduced-motion friendly interactions.
  • High-contrast colors for text and interactive elements.
  • Semantic HTML landmarks for navigation, main content, and footer.

Ongoing Improvements

We audit new content for accessibility and are working to improve form labels, color contrast testing, and support for assistive technologies.

Report a Barrier

If you encounter an accessibility issue, please let us know so we can fix it promptly.

  • Email: nursesunderfire1@gmail.com
  • Response target: within five business days.
  • Please include the page URL, assistive technology used, and a brief description of the issue.

Alternative Formats

We can provide key content in alternative formats (plain text, large print, or simplified summaries) upon request. Reach out via email and note your preferred format.
